# Agenda de Contatos

Uma aplicação web simples para gerenciar contatos telefônicos.

## Funcionalidades

-  Cadastro de contatos com nome e telefone
-  Visualização em tabela
-  Exclusão de contatos
-  Persistência de dados com localStorage
-  Design responsivo
-  Validação de campos

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage para armazenamento local

## Como usar

1. Preencha o nome e telefone no formulário
2. Clique em "Cadastrar Contato"
3. O contato aparecerá na tabela
4. Use o botão "Excluir" para remover um contato
